# Schemarama - Multi-Platform Structured Data Validation

This repository provides ShEx validation schemas for Schema.org structured data
against multiple search engine and platform requirements.

## Supported Platforms

- **Google**: 13 types, 64+ validation shapes
- **Pinterest**: 3 Rich Pin types
- **Bing**: 5 core types

## Deployment

This site is automatically deployed to GitHub Pages via GitHub Actions.
